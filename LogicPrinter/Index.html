What is this website for?

Logical equations usually have many special symbols, like → for conditional statement, ↔ for biconditional statement, ∨ for or statement, etc. These symbols are usually not found on keyboard, and you would typically have to rely on remembering obscure alt codes or installing heavy software with strange names for characters.

This website plans to fix that!

How does it work?

This website uses Regex to convert statements like "AND", "OR", "THEN", "IFF", "EQUALS", "FOR EVERY", "FOR SOME" into their respective logical symbols. If you do not want statements to be converted, just put an escape character before it (Example: \AND) (Might not work for Safari Browser). This makes converter ignore the escaped word.

Remember that the word shortcuts for the statements have to be in all capital letters to be converted!

There are also typable alternate symbols for them too, some of which are common in many programming languages like "&&" for AND
